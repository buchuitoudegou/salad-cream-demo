## 注释规范
### 文件头部注释
```
/**
 *
 * @description 文件用途描述
 * @author your-name
 * @created created date
 * @updated updated date
 *
 */

```
### 函数注释
```
/**
 *
 * 函数描述
 * @author your-name
 * @params {参数类型} 参数名 描述
 * @params ...
 * @return {返回类型} 描述
 *
 */
```
